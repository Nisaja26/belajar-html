Ringkasan Bab

  1. Elemen head adalah wadah untuk metadata (data tentang data)
  2. Elemen head ditempatkan di antara html tag dan body tag
  3. Elemen ini title diperlukan dan menentukan judul dokumen
  4. Elemen ini style digunakan untuk menentukan informasi gaya untuk satu dokumen
  5. Tag ini link paling sering digunakan untuk menghubungkan ke lembar gaya eksternal
  6. Elemen ini biasanya digunakan untuk menentukan set karakter, deskripsi halaman, kata kunci, penulis dokumen, dan pengaturan viewport
  7. Elemen ini script digunakan untuk mendefinisikan JavaScript sisi klien
  8. Elemen ini menentukan URL dasar dan/atau target untuk semua URL relatif di suatu halaman
