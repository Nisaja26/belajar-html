HTML Quotation and Citation Elements

HTML 'blockquote' untuk Kutipan

Elemen HTML 'blockquote' mendefinisikan bagian yang dikutip dari sumber lain.Browser biasanya membuat indentasi elemen 'blockquote'.

Berikut kutipan dari situs WWF:

Selama 60 tahun, WWF telah berupaya membantu manusia dan alam untuk berkembang. Sebagai organisasi konservasi terkemuka di dunia, WWF bekerja di hampir 100 negara. Di setiap tingkat, kami berkolaborasi dengan orang-orang di seluruh dunia untuk mengembangkan dan memberikan solusi inovatif yang melindungi komunitas, satwa liar, dan tempat tinggal mereka.

HTML 'q' untuk Kutipan Singkat

Tag HTML 'q' mendefinisikan kutipan singkat.

Tujuan WWF adalah untuk: Membangun masa depan di mana masyarakat hidup selaras dengan alam.

HTML 'abbr' untuk Singkatan

Tag HTML 'abbr' mendefinisikan singkatan atau akronim, seperti "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Menandai singkatan dapat memberikan informasi berguna bagi browser, sistem terjemahan, dan mesin pencari.
Tip: Gunakan atribut judul global untuk menampilkan deskripsi singkatan/akronim saat Anda mengarahkan mouse ke elemen.

WHO didirikan pada tahun 1948.

HTML 'address' untuk Informasi Kontak

Tag HTML 'address' mendefinisikan informasi kontak penulis/pemilik dokumen atau artikel. Informasi kontak dapat berupa alamat email, URL, alamat fisik, nomor telepon, alamat media sosial, dll. Teks dalam elemen 'address' biasanya ditampilkan dalam huruf miring, dan browser akan selalu menambahkan baris baru sebelum dan sesudah elemen 'address'.

HTML 'cite' untuk Judul Karya

Tag HTML 'cite' mendefinisikan judul karya kreatif (misalnya buku, puisi, lagu, film, lukisan, patung, dll.). Catatan : Nama seseorang bukanlah judul suatu karya. Teks dalam elemen 'cite' biasanya ditampilkan dalam huruf miring.

Jeritan oleh Edvard Munch. Dilukis pada tahun 1893.

HTML 'bdo' untuk Penggantian Dua Arah

BDO adalah singkatan dari Bi-Directional Override.

This text will be written from right to left
