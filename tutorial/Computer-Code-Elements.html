Computer Code Elements HTML

HTML berisi beberapa elemen untuk mendefinisikan masukan pengguna dan kode komputer.

x = 5; y = 6; z = x + y;

HTML kbd Untuk Input Keyboard

Elemen HTML digunakan untuk menentukan input keyboard. Konten di dalamnya ditampilkan dalam font monospace default browser.

Save the document by pressing Ctrl + S

HTML Untuk Output Program

Elemen HTML digunakan untuk menentukan contoh keluaran dari program komputer. Konten di dalamnya ditampilkan dalam fon monospace bawaan browser

pesan untuk komputer Nissa

File not found.
Press F1 to continue

HTML Untuk Kode Komputer

Elemen HTML digunakan untuk mendefinisikan sepotong kode komputer. Konten di dalamnya ditampilkan dalam font monospace bawaan browser.

x = 5; y = 6; z = x + y;

Perhatikan bahwa elemen tersebut tidak mempertahankan spasi tambahan dan jeda baris.

Untuk memperbaikinya, Anda dapat meletakkan elemen di dalam 

elemen:
     
    
    
        
        x = 5;
        y = 6;
        z = x + y;
        
    

          
    
HTML  Untuk Variabel

    
Elemen HTML digunakan untuk mendefinisikan variabel dalam pemrograman atau dalam ekspresi matematika. Konten di dalamnya biasanya ditampilkan dalam huruf miring.
    
    
    
The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.
